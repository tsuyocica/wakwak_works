# アプリケーション名

WakWakWorks

# アプリケーション概要

施工管理者と作業員をマッチングするアプリです。施工管理者は、案件を投稿し、作業内
容・募集人数・報酬・作業日程を設定できます。作業員は案件を閲覧し、チャットを通じ
て応募。施工管理者が承認するとマッチングが成立します。リアルタイムのチャット機能
も搭載し、スムーズなやり取りが可能。短期・ 単発の仕事探しにも最適な、建設業界向
けのマッチングサービスです。
